# Security Policy

## Status

Aegis is **not production-ready**. It is early-stage scaffolding with no encryption or
key management implemented yet. Do not use it to protect sensitive data.

## Reporting a vulnerability

Please email security reports to:

- (SOON)

If you are unsure whether an issue is security-related, you may still report it
through the same channel. Do not open public issues for vulnerabilities.

## What to include

- A clear description of the issue
- Steps to reproduce or a proof of concept
- Affected version/commit, if known
- Suggested fixes or mitigations, if available
